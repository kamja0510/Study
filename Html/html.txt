●HTML (Hypertext Markup Language)

   - HTML
      ㄴ HyperText : 하이퍼링크를 통해 어떤 문서에서 다른 문서로 이동 할 수 있는 텍스트
      ㄴ Makrup : (컨텐츠를)표시하다
      ㄴ Language : 언어
      
      => 기본적인 웹 언어 중, 특히 하이퍼텍스트를 작성하기 위해 개발되었다.

   - 브라우저한테 웹사이트 상에 어떤 내용(컨텐츠)이 들어갈지를 정의해준다.

   - HTML 코드가 웹브라우저를 통해 해석되어 표현되는 과정을 렌더링이라 한다.

   - Public Domain ( 어떠한 저작권도 존재하지 않음. )

   - 구조 : <name(이름) attribute(속성) = "value(값)">contents</name>

   # 공부하면서 기억할 만한 것들.

      1. 크롬에서 ctrl + o 누르고 html 파일 열어서 페이지 열기. 
      2. 한글이 깨지면 <meta charset = "uft-8"> 추가 해보기. 우리가 파일을 저장하면 UTF-8 로 저장이 되기 때문에 웹브라우저에게 UTF-8로 페이지를 열라고 해주어야 한다.
      3. 블록태그와 인라인태그
      4. 태그 중첩 규칙 - 블록태그 내부에는 블록태그와 인라인태그가 중첩될 수 있다, 인라인태그 내부에는 인라인태그만 중첩될 수 있다, 문단 태그는 블록태그이지만 인라인태그만 중첩될 수 있다.
      5. 웹 컨텐츠 접근성 지침을 따라 웹을 만들어야함.
      6. 웹 표준 : 웹에서 표준으로 사용 되는 기술이나 규칙. 구조(Markup), 표현(Style), 동작(Script)으로 구분해 사용하는 것.
