● HTML (Hypertext Markup Language)

   - HTML 문법은 딱 한가지 : <name(태그) attribute(속성) = "value(값)"> contents </name>

   # 공부하면서 기억할 만한 것들.

      1. 크롬에서 ctrl + o 누르고 html 파일 열어서 페이지 열기. 
      2. 블록 레벨 요소는 블럭 공간 자체를 차지하는 요소, 인라인 요소는 컨텐츠를 표기하기 위한 공간만 차지하는 요소.
      3. 태그 중첩 규칙 - 블록 레벨 요소 내부에는 블록 레벨 요소와 인라인 요소가 중첩될 수 있다, 인라인 요소 내부에는 인라인 요소만 중첩될 수 있다, 문단 태그는 블록 레벨 요소이지만 인라인 요소만 중첩될 수 있다.
      4. 웹 컨텐츠 접근성 지침을 따라 웹을 만들어야함.
      5. 웹 표준 : 웹에서 표준으로 사용 되는 기술이나 규칙. 구조(Markup), 표현(Style), 동작(Script)으로 구분해 사용하는 것.
      6. 텍스트 컨텐츠를 작성할 때 띄어쓰기는 한번에 한칸만 알아듣고, 엔터를 이용한 줄바꿈은 알아듣지 못한다.
