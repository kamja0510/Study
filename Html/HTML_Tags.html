• <!DOCTYPE html>

    • - 유효한 문서 형식을 나타내기 위해 선언하는 태그.

  • <html></html>

    • - 전체 페이지의 컨텐츠를 포함하는 기본 태그.

  • <head></head>

    • - 메타데이터를 담는 태그.

    • <title></title>

      • - 페이지 제목이 설정되며 페이지가 로드되는 브라우저 탭에 제목으로 표시해주는 태그.

    • <meta>

    • <link>

    • <style>

    • <script>

  • <body></body>

  • <footer></footer>
